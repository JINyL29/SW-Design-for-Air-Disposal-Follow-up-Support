# 📅 2026-06-04 — 드론 전기 계통 고장 진단 시스템 UI 개선

## 1. Dashboard UI 개선 (`csc_06_dashboard_ui/`)

### 정비 절차 메시지 가독성 향상
- 정비 절차 헤더/본문 폰트 10px → 12px로 확대
- 부품명 11px → 14px, 상태/심각도 텍스트 10px → 11px
- 모든 긴 텍스트에 `setWordWrap(True)` 적용하여 잘림 방지

### 왼쪽 패널 구조 변경
- "고장 수" KPI 카드 제거 (우측 패널에서 이미 상세 확인 가능하므로 중복 제거)
- 좌측 패널 폭 220px → 190px 축소 → 중앙 배선도 영역 확장
- 제거된 자리에 **시스템 건강도 게이지** 추가 (정상 부품 비율 % 표시, 80% 이상 녹색 / 50% 이상 노랑 / 이하 빨강으로 동적 색상 변경)

### 우측 고장 상세 패널 기능 추가
- 패널 폭 280px → 300px 확대
- 헤더에 **심각도 필터 버튼** 추가 (전체 / 긴급 / 높음 / 중간)
- 각 고장 카드에 **감지 시각 표시** 추가 (HH:MM:SS)
- 타이틀 바 우측에 **마지막 진단 시각 + 부품 ID** 실시간 표시

## 2. Virtual Fault Injection & Diagnostic Simulation UI 개편 (`csc_07_operator_control_ui/`)

### 이름 전면 변경
- 기존: `Operator Control Panel`
- 변경: `Virtual Fault Injection & Diagnostic Simulation UI`
- 창 제목, 내부 타이틀 레이블, 섹션명, `__init__.py` 주석 전부 반영

### 창 크기 및 해상도 확대
- `setFixedSize(640, 860)` → `setMinimumSize(860, 900)` + `resize(980, 1000)`
- 고정 크기 → 사용자 자유 조절 가능

### UI 요소 전반 업스케일
- 부품 목록 최소 높이 180px → 260px, 아이템 폰트 12px, 패딩 확대
- 전압 입력 필드 폰트 13px → 14px
- 시나리오 버튼 폰트 11px → 12px, 패딩 및 테두리 강화
- 진단 결과 프리뷰 값 폰트 14px → 16px
- 정격 전압 참조 폰트 9px → 11px

### 텍스트 색상 버그 수정
- 부품 목록 아이템이 흰색으로 보이던 문제 → `color:#1A2B3C` 명시
- 전압 입력값이 안 보이던 문제 → `color:#1A2B3C` 명시
