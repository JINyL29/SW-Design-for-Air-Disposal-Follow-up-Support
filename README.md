# Project Title : Drone Electrical System Fault Diagnosis and Maintenance Support Software

## Overview

This project aims to design a fault diagnosis and maintenance decision-support software for drone electrical systems.

The system monitors electrical components in a drone platform and generates fault codes based on abnormal electrical conditions such as under-voltage and over-voltage states.

The software provides a maintenance dashboard UI that allows users to:

- Monitor electrical component status
- Detect abnormal voltage conditions
- Generate and manage fault codes
- Support maintenance decision-making

---

## System Architecture

The proposed system consists of three main stages:

### Input
- Component rated voltage database
- Real-time voltage sensor data

### Processing
- Voltage condition analysis
- Under-voltage / Over-voltage detection
- Fault code generation
- Component importance analysis

### Output
- Maintenance dashboard UI
- Fault code visualization
- System status monitoring
